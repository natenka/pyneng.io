# Список (List)

Список у Python це:

* послідовність елементів, розділених між собою комою та укладених у квадратні дужки
* змінюваний упорядкований тип даних


```python
vlans = [10, 20, 30, 77]
commands = ["interface Gi0/1", "ip address 10.1.1.1 255.255.255.0"]
list3 = [1, 20, 4.0, 'word']
```
